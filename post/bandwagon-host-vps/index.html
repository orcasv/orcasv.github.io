BandwagonHost 简单介绍

因其官网网站标识是BandwagonHost，有点类似BanWaGong的拼写，所以我们喜欢称作为搬瓦工。搬瓦工VPS是一款性价比比较高的便宜VPS主机，且适合入门级网友学习Linux和建站用途.

（申明和建议：严禁利用VPS、服务器用于违规用途，我们必须正规使用）

搬瓦工VPS，隶属于美国IT7公司旗下的一款低价OpenVZ VPS主机方案、2017年新增KVM VPS架构，尤其是8款便宜年付VPS，无论从性价比还是稳定性都非常适合大众VPS用户需求，我们可以用来建站、Linux系统学习、软件调试，也可以完成各种初学者需要测试项目正规用途使用。 反正完全可以用作新手学习VPS主机选择性价比VPS使用。相继增加CN2线路、且在2017年10月份新增香港机房。

2019年5月13日 更新, 现在新购买的VPS找不到这个接口了, 那就手动搭建吧, 很快的相信我.

五分钟快速搭建GO版shadowsocks:

// 新建文件夹
mkdir  /usr/local/bin/shadowsocks 

cd  /usr/local/bin/shadowsocks
// 下载shadowsocks Golang二进制文件
wget <https://github.com/shadowsocks/shadowsocks-go/releases/download/1.2.1/shadowsocks-server.tar.gz

tar -xzvf shadowsocks-server.tar.gz

vim ss-config.json     // 新建配置文件并修改

//配置文件内容

{
     "server":"127.0.0.1",	    // 修改为自己服务器IP
     "server_port":8388,		// 自定义端口, 推荐443 
     "local_port":1080,
     "local_address":"127.0.0.1",
     "password":"barfoo!",		// 密码尽量修改复杂一些
     "method": "aes-128-cfb",   // 推荐这个, 已经很安全了
     "timeout":600
 }

// 安装客户端程序, 教程见后面

// 客户端安装好后添加服务器完成后, 测试一下:

shadowsocks-server -c  ss-config.json

// 电脑浏览器或者手机打开 Google.com 看看能否进入, 如果配置正确的话是没有问题的

// 添加shadowsocks到服务, 使其可以开机启动

// 进入目录:

cd /etc/systemd/system/

// 新建shadowsocks服务配置
    
vim  shadowsocks.service

//  填入以下内容
[Unit]
Description=Shadowsocks

[Service]
TimeoutStartSec=0

ExecStart=/usr/local/shadowsocks/shadowsocks-server -c /usr/local/shadowsocks/ss-config.json

[Install]
WantedBy=multi-user.target

// 使服务生效
systemctl enable shadowsocks

// 查看服务状态
systemctl status shadowsocks

// 重启服务
systemctl restart shadowsocks
